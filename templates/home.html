About the Project

Our project leverages machine learning to analyze and predict customer satisfaction based on various factors gathered from in-flight experiences.

Technologies Used

  • Flask: Web application framework for Python
  • SQLite3: Lightweight database for storing survey data
  • Python: Backend logic and machine learning integration
  • NumPy & Pandas: Data analysis and manipulation
  • Matplotlib & Seaborn: Data visualization
  • Joblib: Model persistence and deployment
